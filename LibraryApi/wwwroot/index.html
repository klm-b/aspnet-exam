Доступное API:

Книги
GET api/books Получение всех книг.
GET api/books/{id} Получение книги по идентификатору.
POST api/books Добавление новой книги.
PUT api/books/{id} Обновление существующей книги.
DELETE api/books/{id} Удаление книги.


Авторы
GET api/books Получение всех авторов.
GET api/authors/{id} Получение автора по идентификатору.
POST api/authors Добавление нового автора.
PUT api/authors/{id} Обновление существующего автора.


Категории
GET api/books Получение всех категорий.
GET api/categories/{id} Получение категории по идентификатору.
POST api/categories Добавление новой категории.
PUT api/categories/{id} Обновление существующей категории.


Запросы
GET api/queries/1 Запрос 1. Вывести полную информацию обо всех книгах.
GET api/queries/2 Запрос 2. Вывести название, год издания, автора и цену учебников по Android.
GET api/queries/3 Запрос 3. Вывести название, год издания и количество задачников по LINQ.
GET api/queries/4 Запрос 4. Вывести автора, название, категорию и стоимость для каждой книги, количество которых от 4 до 6.
GET api/queries/5 Запрос 5. Вывести фамилии и инициалы авторов и количество книг этих авторов.
GET api/queries/6 Запрос 6. Для категорий книг вывести количество, минимальную стоимость книги, среднюю стоимость книги, максимальную стоимость книги.
GET api/queries/7 Запрос 7. Для всех книг автора Абрамян М.Э. увеличить стоимость книг на 15%.
GET api/queries/8 Запрос 8. Уменьшить количество книг по C++ на 1, не допускать отрицательных значений.
